# Sistema de Cadastro de Convidados

Projeto simples em Python desenvolvido para armazenar e organizar uma lista de convidados.

## Funcionalidades
- Limite máximo de convidados
- Verificação de maioridade
- Validação de entradas
- Listagem formatada dos convidados

## Tecnologias e Conceitos Utilizados
- Python 3
- Estruturas de repetição e decisão (`while`, `if/elif/else`)
- Listas compostas
- Tratamento de erros com `try/except`
- Formatação avançada de strings
